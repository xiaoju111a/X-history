# X History

[English](#english) | [中文](#中文)

---

## English

A Chrome extension that automatically saves tweets you click into.

### Features

- Auto-save tweets when you visit tweet detail pages
- Local storage (no cloud, no tracking)
- Search through saved tweets
- Export to JSON
- Delete individual tweets or clear all

### Installation

1. Download or clone this repository
2. Open Chrome and go to `chrome://extensions/`
3. Enable "Developer mode" (top right)
4. Click "Load unpacked" and select the `x-history-extension` folder

### Usage

1. Browse X (Twitter) normally
2. Click on any tweet to view its detail page - it will be saved automatically
3. Click the extension icon to view your saved tweets

## 中文

一个自动保存你点击查看的推文的 Chrome 扩展。

### 功能

- 访问推文详情页时自动保存
- 本地存储（无云端、无追踪）
- 搜索已保存的推文
- 导出为 JSON
- 删除单条或清空全部

### 安装

1. 下载或克隆此仓库
2. 打开 Chrome，访问 `chrome://extensions/`
3. 开启右上角「开发者模式」
4. 点击「加载已解压的扩展程序」，选择 `x-history-extension` 文件夹

### 使用

1. 正常浏览 X (Twitter)
2. 点击任意推文进入详情页，会自动保存
3. 点击扩展图标查看已保存的推文
